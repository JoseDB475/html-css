BOX_MODEL

Permite dar estructura y dimensiones a los elementos de una pagina web

Sirve para gestionar el diseño (layout), espaciado, bordes y dimensiones de los elementos, compuesto por: Contenido (texto/imagen), Padding (relleno interior), Borde y Margen (espacio exterior)

DISPLAY

Abarca desde visualizaciones de datos en pantallas hasta interfases de usuarios complejas

En programación, "display" se refiere principalmente a dos contextos: el diseño web (CSS) para controlar la visualización y maquetación de elementos, o componentes electrónicos (hardware/electrónica) como displays de 7 segmentos para mostrar información visual. En CSS, define cómo se comporta una caja, ya sea como bloque, en línea, flexible o cuadrícula.

FLEX_BOX

Permite alinear,distribuir espacio y organizar elementos hijos dentro de un contenedor

Flexbox es un modelo de maquetación CSS3 unidimensional que permite organizar elementos en filas o columnas de forma flexible, facilitando la alineación y distribución del espacio en interfaces web. Sirve para crear diseños adaptables (responsivos) sin usar técnicas complejas, logrando que los elementos ajusten su tamaño y posición automáticamente según la pantalla.

MEDIDAS

Definen el tamaño de los elementos dentro de un lenguaje de etiquetado como html

Las medidas en CSS definen el tamaño, espaciado y disposición de elementos web (texto, contenedores, imágenes), permitiendo controlar el diseño visual. Sirven para crear sitios web adaptables (responsive) a diferentes pantallas, asegurando que elementos como márgenes, rellenos y fuentes mantengan proporciones correctas en móviles y escritorio, utilizando unidades relativas o absolutas.

PROPIEDADES

HTML define la estructura semantica y css aplica propiedades visuales de una pagina web

Las propiedades en CSS son características que definen el diseño, estilo y comportamiento de elementos HTML, como colores, tamaños, bordes o disposición. Sirven para separar el contenido (HTML) de la presentación (CSS), permitiendo crear sitios web atractivos, responsivos y fáciles de mantener al controlar la apariencia.

SELECTORES

Se utilizan en css para seleccionar y aplicar estilos, tamaños, disposicion a los elementos de html

Los selectores en CSS son patrones que permiten identificar y seleccionar elementos HTML específicos (etiquetas, clases, IDs) para aplicarles estilos visuales, como color, tamaño o disposición. Sirven para vincular la estructura (HTML) con la apariencia (CSS), permitiendo diseñar, organizar y hacer interactiva una página web.
